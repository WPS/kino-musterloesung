# Glossar

Die Begriffsdefinitionen können sich je Subdomäne unterscheiden.

## Filmauswahl

| Begriff | Definition |
|---------|------------|
| **Film** | Ein Kinofilm, der ins Programm aufgenommen und gezeigt werden kann. |
| **Kinoprogramm** | Übersicht, welche Filme an welchen Tagen zu welchen Zeiten gezeigt werden. |
| **Saal** | Ein Vorführraum im Kino, in dem Filme gezeigt werden. |
| **Vorstellung** | Eine konkrete Aufführung eines Films zu einer bestimmten Zeit in einem bestimmten Saal. |

## Kartenverkauf

| Begriff | Definition |
|---------|------------|
| **Beginn** | Datum und Uhrzeit, zu denen eine Vorstellung anfängt. |
| **Film** | Der Filmtitel einer Vorstellung. Im Kartenverkauf zählt nur, *welcher* Film gezeigt wird — Laufzeit, Besetzung etc. sind hier nicht relevant. |
| **Geldbetrag** | Ein Geldwert mit Währung. |
| **Kinokarte** | Berechtigt zum Besuch einer bestimmten Vorstellung auf einem bestimmten Platz. |
| **Platz** | Ein Sitzplatz im Saal, der für eine Vorstellung frei oder verkauft ist. |
| **Platzanzahl** | Wie viele Plätze ein Kunde für eine Bestellung wünscht. |
| **PlatzKategorie** | Lage eines Platzes im Saal: *Parkett* (regulär) oder *Loge* (mit Aufpreis). |
| **Saal** | Der Saal, in dem eine Vorstellung stattfindet. |
| **Saalplan** | Die Sitzordnung einer Vorstellung mit dem aktuellen Belegungsstand jedes Platzes. |
| **Sitzplatzvergabe** | Das Finden, Reservieren und Verkaufen freier Plätze für eine Vorstellung. |
| **Vorstellung** | Eine bestimmte Aufführung, für die Karten verkauft werden. |
| **VorstellungKategorie** | Preisstufe einer Vorstellung: *Standard*, *PrimeTime* oder *Kinderfilm*. |
| **Zahlungsstatus** | Stand einer Zahlung: *Ausstehend*, *Eingegangen* oder *Abgebrochen*. |
| **Zahlungsvorgang** | Die Bezahlung einer Kartenbestellung. |
| **ZusammenhängendePlätze** | Eine Reihe nebeneinanderliegender freier Plätze, die gemeinsam gebucht werden können. |
